Generate and manage QR codes that redirect to specific views in your Django project

## Installation

Install this via pip (or your favourite package manager):

`pip install django-qr`

Add the app to your `INSTALLED_APPS`:

```python
INSTALLED_APPS = [
    # ...
    "django_qr",
]
```
